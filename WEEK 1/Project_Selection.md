## Selected Projects

### Project 2: Regression for Price Prediction
**Real-world problem:** Businesses need to estimate values like house prices based on
property characteristics.
**Why I chose it:** Regression is a foundational supervised learning technique and
directly builds on concepts from Project 1. It also lets me practice feature
engineering, cross-validation, and hyperparameter tuning — all core ML skills.
**Tools:** Linear Regression, Random Forest, Scikit-learn, GridSearchCV

### Project 3: Customer Clustering
**Real-world problem:** Marketing teams need data-driven customer segments to
target campaigns effectively, rather than treating every customer the same way.
**Why I chose it:** This project introduces unsupervised learning, which is a
different paradigm from regression/classification. It rounds out my understanding
of both major branches of ML and requires interpreting results without ground-truth
labels — a valuable, distinct skill.
**Tools:** K-Means, DBSCAN, PCA, Scikit-learn
